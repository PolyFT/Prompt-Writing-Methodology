# 合规 Prompt 示例（科研规范版）

---

## 说明

本文件用于给出符合  
**《科研场景下面向大模型的提示词编写方法学》**  
的提示词（Prompt）示例，供团队成员在科研写作与科研项目文本中直接参考与复用。

所有示例均面向**严肃科研场景**，强调学术语义可控、概念边界清晰与输出结果可直接使用。

---

## 示例一：SCI 论文引言段落生成（合规）

你正在协助完成一篇面向 SCI 期刊投稿的科研论文写作任务。

以下内容属于论文引言部分，用于阐述研究背景与问题定位。  
请在不引入新的研究对象、不扩展理论框架、不提升结论层级的前提下，  
对我提供的研究设定进行学术化表述。

写作要求如下：  
1. 保持严格的学术语气，不使用评价性或宣传性语言；  
2. 表述仅限于我已明确陈述的研究问题与研究动机；  
3. 不得引入新的概念、变量或机制性解释；  
4. 形成一段约 200 字的连续论述文本；  
5. 输出内容需可直接用于论文正文，无需再次修改。

以下为研究设定文本：  
【此处粘贴你的研究背景原文】

### 合规点说明（内部理解用）

- 明确学术场景（SCI 引言）  
- 锁定语义来源（仅限已给内容）  
- 明确输出尺度与完成状态  
- 禁止模型自主扩展

## 示例二：结果与讨论段落重述（合规）

当前任务为科研论文中“结果与讨论”部分的文字重述。

请在严格基于我提供的实验结果描述的前提下，  
对相关现象进行学术化、逻辑更清晰的表述。  
本任务不涉及新的数据解释或机理推断。

具体要求如下：  
1. 仅对实验现象进行描述性与比较性表述；  
2. 不得将现象性结果上升为因果性或机制性结论；  
3. 不得引入实验中未测试的变量；  
4. 语言需符合 SCI 期刊写作规范；  
5. 输出为 1–2 个连续自然段。

实验结果原文如下：  
【此处粘贴实验结果文字或图注说明】

## 示例三：方法学段落整理（合规）

你正在协助整理科研论文中的“方法学”描述文本。

请在保持技术细节完整、不改变实验逻辑顺序的前提下，  
对我提供的方法描述进行语言规范化处理。  
本任务仅涉及表述重构，不涉及方法改进或优化建议。

具体要求：  
1. 严格使用学术性、客观描述语言；  
2. 不得添加新的实验步骤或参数；  
3. 不得使用总结性或评价性语句；  
4. 输出为可直接用于论文 Methods 部分的连续文本。

方法原始描述如下：

## 示例四：图注（Figure Caption）生成（合规）

当前任务为科研论文中图注文字的规范化生成。

3. 使用客观、中性、描述性语言；  
4. 输出为单段图注文本。

当前任务为科研项目文本中的方法学总结段落撰写。

2. 不引入超出当前项目范围的研究方向；  
3. 输出为约 300 字的连续论述文本；  
4. 语言风格符合科研项目申报文本要求。
